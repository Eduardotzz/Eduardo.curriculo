🌎 World Guessing Game – Ranking Edition

Projeto educacional desenvolvido em Python aplicando Programação Orientada a Objetos (POO) e arquitetura MVC. O jogo utiliza a API REST Countries para desafiar o jogador a descobrir países com base em dicas como Capital e Região.

A cada acerto, pontos são acumulados e, ao final, a pontuação é salva em um ranking Top 5 persistente com data e hora.

🚀 Tecnologias: Python, Tkinter, SQLite3, Requests, API REST Countries
